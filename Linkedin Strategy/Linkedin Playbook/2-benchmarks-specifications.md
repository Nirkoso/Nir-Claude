## 2. Ad Format Specifications

### Single Image Ads

| Specification | Requirement | Optimal |
|---------------|-------------|---------|
| Image size | 1200x627px | Exactly 1200x627 |
| File size | <5MB | 1-2MB |
| Format | PNG or JPG | PNG for graphics, JPG for photos |
| Headline | 70 chars max | 25-50 chars |
| Description | 150 chars max | 70-100 chars |
| Text overlay | <20% coverage | Minimal or none |

**Best practices:**
- High contrast visuals
- Include faces (+38% engagement)
- Brand logo visible but not dominant
- Minimal text overlay

### Carousel Ads

| Specification | Requirement | Optimal |
|---------------|-------------|---------|
| Card count | 2-10 cards | 3-5 cards |
| Image size (per card) | 1080x1080px | Exactly 1080x1080 |
| File size (per card) | <5MB | 1-2MB |
| Headline (per card) | 45 chars max | 30-40 chars |
| Description (per card) | 255 chars max | 150-200 chars |
| Intro text | 255 chars max | 150-200 chars |

**Best practices:**
- First card must hook (strongest value prop)
- Each card delivers standalone message
- Final card includes clear CTA
- Consistent visual style across cards

### Video Ads

| Specification | Requirement | Optimal |
|---------------|-------------|---------|
| Length | 3 seconds - 30 minutes | 15-30 seconds |
| Dimensions | 1080x1080 or 1920x1080 | 1080x1080 (square) |
| Format | MP4 | MP4 |
| File size | <200MB | <50MB |
| Intro text | 255 chars max | 150-200 chars |

**Critical requirements:**
- First 3 seconds must hook
- Captions required (90% watch muted)
- Brand visible throughout
- Custom thumbnail strongly recommended

### Lead Gen Forms

| Element | Specification | Optimal |
|---------|---------------|---------|
| Pre-filled fields | Name, Email, Company, Job Title | Use all available |
| Custom fields | 0-10 | 3-5 max |
| Subject line | 60 chars | 40-50 chars |
| Privacy policy | Required | Clear, concise |
| Thank-you message | Custom | Include next steps |

**Impact of field count:**
- 3 fields: ~12% conversion
- 5 fields: ~10% conversion (10% drop per field)
- 7 fields: ~8% conversion
- 10+ fields: <5% conversion

### Message Ads (Sponsored InMail)

| Specification | Requirement | Optimal |
|---------------|-------------|---------|
| Subject | 60 chars max | 40-50 chars |
| Body | 1,500 chars max | 500-700 chars |
| CTA button | 25 chars | 15-20 chars |
| Sender info | Required | Name, title, company |

**Best practices:**
- Personalize with {FIRSTNAME}
- Conversational tone
- Single clear CTA
- Only delivered when recipient active

### Document Ads

| Specification | Requirement | Optimal |
|---------------|-------------|---------|
| Format | PDF or PPT | PDF |
| Pages | Max 300 | 5-15 pages |
| File size | Max 100MB | <10MB |
| First page design | Critical | Acts as thumbnail/hook |

**Best practices:**
- Design for mobile viewing
- Clear chapter titles
- CTA on final page
- High-value content only

## 4. Audience Sizing Guidelines

### Optimal Ranges

| Audience Size | Status | Performance | Action |
|---------------|--------|-------------|--------|
| <30K | Critical | Very poor delivery | Expand immediately |
| 30K-50K | Marginal | Poor delivery, high CPL | Enable 15-20% expansion |
| **50K-150K** | **Optimal** | **Stable, efficient** | **Ideal for conversion** |
| **150K-500K** | **Optimal** | **Best for scale** | **Sweet spot** |
| 500K-1M | Large | Good for awareness | May reduce relevance |
| >1M | Very Large | Awareness only | Too broad for conversion |

### Audience Expansion Impact

| Expansion % | Audience Increase | CTR Impact | CPL Impact |
|-------------|------------------|------------|------------|
| 0% (disabled) | Baseline | Baseline | Baseline |
| 10% | +10-15% | -0-5% | -10-15% (improves) |
| 15% | +15-20% | -5-8% | -15-20% (improves) |
| 20% | +20-30% | -8-10% | -20-25% (improves) |

**Recommendation:** Always enable 10% minimum, 15-20% if audience <100K.

## 7. Time-Based Benchmarks

### Campaign Learning Curve

| Week | Expected CPL | Expected CTR | Status | Action |
|------|-------------|--------------|--------|--------|
| Week 1 | 50-100% above target | 0.25-0.35% | Learning | Monitor, don't optimize |
| Week 2 | 20-50% above target | 0.30-0.45% | Stabilizing | First optimization pass |
| Week 3 | 10-30% above target | 0.35-0.50% | Optimizing | Fine-tune, test variants |
| Week 4+ | At or below target | 0.40-0.60%+ | Steady state | Scale or maintain |

### Creative Lifespan

| Ad Type | Lifespan | Refresh Trigger | Replacement Strategy |
|---------|----------|----------------|---------------------|
| Single Image | 6-8 weeks | CTR -30% from peak | New image + headline |
| Carousel | 8-10 weeks | CTR -25% from peak | Reorder cards, new content |
| Video | 8-12 weeks | View rate -40% | New video or edit existing |
| Message Ad | 4-6 weeks | Open rate -35% | New subject + body |

### Frequency Thresholds

| Frequency | Status | Action |
|-----------|--------|--------|
| <1.5 | Healthy reach | Room to scale |
| 1.5-2.5 | Optimal | Maintain |
| 2.5-3.5 | Elevated | Monitor closely |
| 3.5-4.5 | High | Refresh creative or expand audience |
| >5.0 | Saturation | Pause immediately |

## Usage Notes

**When setting targets:**
- Use industry vertical benchmarks (Section 1)
- Adjust for company stage (early/growth/mature)
- Factor in deal size economics (Section 6)
- Account for learning curve (Week 1 vs. Week 4)

**When diagnosing issues:**
- Compare to performance rating scale (Section 1)
- Check against ad format specs (Section 2)
- Verify audience size is optimal (Section 4)
- Review funnel drop-offs (Section 5)

**When reporting:**
- Use benchmarks as context, not guarantees
- Note: Results vary by targeting, creative, offer quality
- Always compare to industry vertical, not generic B2B
